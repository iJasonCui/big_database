README for the SyBooks™ CD-ROM
version 2.0


What's On This CD
-----------------

Product documentation is provided on the SyBooks™ CD in a platform-independent bookshelf that contains fully searchable, HTML-based documentation and PDF files.
 
Some documentation is provided in PDF files. To view the PDF files, you must have Acrobat Reader installed on your machine. Go to http://www.adobe.com to download Acrobat Reader. PDF files can be accessed through the PDF directory on the SyBooks CD.

You can also access all documentation on the Sybase Manuals website (www.sybase.com/support/manuals).


Installation Notes
------------------
On Windows, the CD should autorun and execute the SyBooks installation. After installation, you can launch the SyBooks bookshelf at any time. Select Start > Sybase > SyBooks and choose the product documentation you want to view. 

To view documentation on UNIX, insert the CD and use the appropriate UNIX version of Acrobat Reader to view the PDF files contained in the PDF directory. 


Troubleshooting
-----------------
If you are unable to view PDF files in the bookshelf on Windows, verify the following in your browser:

1. Active X settings must be enabled.


Uninstalling SyBooks
--------------------
1. Initiate the InstallShield wizard. On Windows, do one of the following:
   a. Select Start |Control Panel | Add/Remove Programs. Select the appropriate SyBooks for <product> and click Remove.
   b. At the command prompt, enter CD <installation directory>\SyBooks\uninstallers\<product> .\uninstall
2. Follow the uninstallation instructions in the InstallShield wizard.
3. Click Finish when the uninstall is successful.
4. Delete any files remaining in the <installation directory> | SyBooks directory.
